Balderdash

A real time multi player game that plays similar to the popular board game by the same name. Players are presented with a obscure word. Each player makes up a definition for that word which they believe the other players will vote for as being correct over the real definition. Ajax requests were used to continually check the game state for all player views and synchronize the views once all definitions or guesses had been submitted. Check out the code on Github!
